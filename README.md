# David Sackey Portfolio

Static GitHub Pages profile for a frontend engineer and full-stack architect.

## Files

- `index.html` - page structure and portfolio content.
- `styles.css` - responsive visual design.
- `script.js` - small footer year update.
- `assets/David_Sackey_Resume.pdf` - downloadable resume.
- `assets/work-screenshots/` - cropped work screenshots captured from the shared CV evidence document.

## Publish

Push this repository to GitHub and enable GitHub Pages for the branch that contains these files. The site is static and does not need a build step.
